(static) createShortTextsNetwork(texts, stopWords, relationThreshold, mode, applyIntensity, varnameopt) → {Network}

builds a network out of a list of short strings, adds a property wordsTable to each node (with words and weights)

Parameters

Name Type Attributes Description
texts StringList
stopWords StringList
relationThreshold Number

threshold to create a relation
mode Number


0:pseudoentropy, by finding key words with low entropy (words occurring in a single text or in all texts have maximum entropy, occuring in 0.25 texts minimum entropy (max weight))
1:originality
2:skewed entropy
3:originality except isolation
applyIntensity Boolean

takes into account occurrences of word into each text
varname Table <optional>

if a words frquency table is provided, les frequent words are weighed

Returns Network
